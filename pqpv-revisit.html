Measured

library(RODBC)
ch <- odbcConnect("PQPV")
M <- sqlQuery(ch,'select 
  logs.`hitmanSWV_m1s-1` as `log.swv`, 
  logs.stemNumber as stem, 
  logs.length_m as `log.length`, 
  logs.logPositionInStem as `log.posn`, 
  stems.`hitmanSWV_m1s-1` as `stem.swv`,
  stems.`D1.5_mm`/10 as `stem.dbh`,
  joe90.barcode,
  joe90.logNumber as `log.number`,
  joe90.r_mm as r,
  joe90.phi_deg as theta,
  joe90.length_m as length,
  joe90.volume_cm3 as volume,
  joe90.mass_kg as mass,
  joe90.resonance_hz as f1,
  joe90.`density_kg1m-3` as den,
  joe90.MOEd_GPa as moe,
  w.*
from 
  (select
     scanId,
     max(if(measure="bow.extr_mm",value,null)) as `b.extr`,
     max(if(measure="bow.max_mm",value,null)) as `b.max`,
     max(if(measure="bow.min_mm",value,null)) as `b.min`,
     max(if(measure="bow.rms_mm",value,null)) as `b.rms`,
     max(if(measure="bow.xref_mm",value,null)) as `b.mid`,
     max(if(measure="crook.extr_mm",value,null)) as `c.extr`,
     max(if(measure="crook.max_mm",value,null)) as `c.max`,
     max(if(measure="crook.min_mm",value,null)) as `c.min`,
     max(if(measure="crook.rms_mm",value,null)) as `c.rms`,
     max(if(measure="crook.xref_mm",value,null)) as `c.mid`,
     max(if(measure="twist.extr_deg",value,null)) as `t.extr`,
     max(if(measure="twist.max_deg",value,null)) as `t.max`,
     max(if(measure="twist.min_deg",value,null)) as `t.min`,
     max(if(measure="twist.rms_deg",value,null)) as `t.rms`,
     max(if(measure="twist.xref_deg",value,null)) as `t.tot`
   from
     warp
   where
     postId=1
   group by 
     scanId
  ) as w,
  joe90, 
  logs,
  stems
where 
  logs.logNumber=joe90.logNumber and
  stems.stemNumber=logs.stemNumber and
  joe90.logNumber=logs.logNumber and
  joe90.scanId=w.scanId and
  joe90.length_m > 4.5 and
  joe90.MOEd_GPa < 20')
M$src <- "measured"

Statistical Testing

None of the board quality distributions are normal which rules aout a great swathe of distribution comparison tests.

Kolmogorov-Smirnoff

The two-sample K-S test is the obvious choice:

T <- data.frame()
for (p in c('den','moe','b.mid','c.mid','t.tot')) {
  ks <- ks.test(M[,p],P[,p],alternative='two.sided')
  T <- rbind(T, data.frame(ks.stat=ks$statistic, ks.p.value=ks$p.value, row.names=p))
}

K-S is most sensitive to differences near the median (http://www.aip.de/groups/soe/local/numres/bookcpdf/c14-3.pdf) we are more interested in differences in the tail(s). Are there better tests?

Instead of testing statistical significance, what bout tests of practical significance?
